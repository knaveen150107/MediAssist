## Overview

MediAssist AI is an intelligent healthcare assistant designed to help users understand complex medical reports without requiring medical expertise.

The platform uses OCR, NLP, and Large Language Models (LLMs) to extract medical information, identify abnormal values, generate simplified explanations, and answer user questions in both English and Tamil.

---

## Key Features

### Medical Report Analysis

- Upload laboratory reports
- Extract important biomarkers
- Detect abnormal values
- Generate health summaries
